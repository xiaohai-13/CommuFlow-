# Emergency Order Change SOP

## 1. Scope
Applies to all emergency order changes affecting production schedule.

## 2. Process
1. Initiator submits emergency change request in Feishu
2. Production dept evaluates capacity impact within 2 hours
3. Procurement dept confirms material availability
4. Sales dept communicates timeline change to customer
5. Production Director gives final approval

## 3. Roles
- Production: capacity assessment
- Procurement: material confirmation
- Sales: customer communication
- Production Director: final approver

## 4. Time Requirement
Emergency order changes must complete full approval within 4 working hours.
